# Pocket

Semantic Versioning Changelog to Pokédex Entry Generator

## Overview

`Pocket` takes your [semantic versioning](https://semver.org/) [changelog](https://en.wikipedia.org/wiki/Changelog) and converts it to a custom Pokédex.
